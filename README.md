# Analise-de-Interacao-de-Clientes-numa-Academia
A rede de academias Model Fitness está desenvolvendo uma estratégia de interação com o cliente baseados em dados analíticos. A Model Fitness digitalizou uma parte dos perfis dos clientes. A tarefa é analisá-los e vir com uma estratégia de retenção de clientes.
Objetivos:
Aprender a predizer a probabilidade de rotatividade (para o mês seguinte) para cada cliente
Elaborar retratos de usuários típicos: selecione os grupos mais marcantes e descreva suas principais características
Analisar os fatores que mais impactam a rotatividade
Conclusoes e Recomendações
Frequência de Visitas é o fator mais importante. 
Clientes que visitam mais vezes têm menor probabilidade de sair.
Tempo de contrato mais longo (6 meses ou 1 ano) se correlaciona com menor rotatividade.
Clientes de empresas parceiras e com promoções de amigos têm maior retenção.
Clientes que participam de aulas em grupo são mais leais."
Recomendações de Retenção
1. Criar um Programa de Engajamento
2. Incentivar visitas regulares com recompensas (pontos, brindes)
3. Alertas automáticos se o cliente faltar 2 semanas
4. Estender os Contratos com Benefícios
5. Descontos ou bônus para migração de planos mensais para trimestrais/semestrais
6. Fortalecer Parcerias, expandindo acordos com empresas para incluir mais funcionários em programas de fidelidade
7. Investir em Aulas em Grupo e Comunidade
8. Aumentar a oferta de aulas coletivas (yoga, HIIT, jump, pump, etc)
9. Promover eventos e desafios comunitários mensais para fortalecer o senso de pertencimento e uniao
